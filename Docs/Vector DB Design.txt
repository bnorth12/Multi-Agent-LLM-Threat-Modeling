# Vector DB Design — Offline Aerospace Threat Modeling

All collections run 100 % offline using **LanceDB** (preferred) or **Chroma** as fallback.

### Embedding Model (locked)
- **Primary**: `BAAI/bge-large-en-v1.5` (runs on internal RTX 6000/4090/A100 GPUs)
- **Backup**: `intfloat/e5-large-v2` (if bge fails licensing check)
- Dimension: 1024
- Normalized embeddings: Yes

### Ingestion Script (Python — runs on air-gapped workstation)
```python
# scripts/ingest_all.py
from langchain_community.embeddings import HuggingFaceBgeEmbeddings
from langchain.vectorstores import LanceDB
import lance
import json, os, glob

embedder = HuggingFaceBgeEmbeddings(
    model_name="BAAI/bge-large-en-v1.5",
    model_kwargs={'device': 'cuda'},
    encode_kwargs={'normalize_embeddings': True}
)

def ingest_json_collection(path: str, collection_name: str, text_field: str):
    docs = []
    for file in glob.glob(f"{path}/*.json"):
        with open(file) as f:
            data = json.load(f)
            for item in data:
                docs.append(Document(page_content=item[text_field], metadata=item))
    vectorstore = LanceDB.from_documents(docs, embedder, table_name=collection_name)
    print(f"Loaded {len(docs)} into {collection_name}")

# Run all
ingest_json_collection("data/attack", "mitre_attack", "description")
# ... repeat for others
